## 📋 Sobre o Projeto
O **Conta Litro** é uma aplicação mobile desenvolvida em **Flutter** que auxilia motoristas no controle e previsão de gastos com combustível. A aplicação recebe dados inseridos pelo usuário (como distância percorrida, consumo médio do veículo e preço do combustível) e retorna uma análise detalhada do consumo esperado para o trajeto.

## ✨ Funcionalidades
- **Cálculo de Consumo:** Estimativa de litros necessários para uma viagem.
- **Interface Intuitiva:** Design focado na experiência do usuário (UX) para entradas rápidas de dados.

## 🛠️ Como Executar o Projeto

1. **Pré-requisitos:**
   - Flutter SDK instalado.
   - Emulador Android/iOS ou dispositivo físico conectado.

2. **Clonar o repositório:**
   ```bash
   git clone [https://github.com/joao-fcosta/conta-litro.git](https://github.com/joao-fcosta/conta-litro.git)
   ```
3. **Instalar dependências:**
   ```bash
   flutter pub get
   ```
4. **Executar a aplicação:**
   ```bash
   flutter run
   ```
